## üöÄ Installation & Setup

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Step 1: Install Required Libraries

Open your terminal/command prompt and run:

```bash
pip install streamlit pandas plotly numpy
```

**Or create a requirements.txt file:**

```text
streamlit==1.31.0
pandas==2.2.0
plotly==5.18.0
numpy==1.26.3
```

Then install using:
```bash
pip install -r requirements.txt
```

### Step 2: Download the Dataset

1. Go to: https://www.kaggle.com/datasets/ankushpanday2/indian-kids-screentime-2025
2. Download the CSV file
3. Keep it in the same folder as your Python script

## üéØ How to Run the Dashboard

### Method 1: Using Command Line

1. **Save the Python code** as `screentime_dashboard.py`

2. **Open terminal/command prompt** in the same directory

3. **Run the command:**
   ```bash
   streamlit run screentime_dashboard.py
   ```

4. **Dashboard will open automatically** in your default browser at `http://localhost:8501`

### Method 2: Using IDE (VS Code/PyCharm)

1. Open the Python file in your IDE
2. Open terminal within IDE
3. Run: `streamlit run screentime_dashboard.py`

### Uploading the Dataset

Once the dashboard opens:
1. Look for the **sidebar** on the left
2. Click on **"Browse files"** under "Upload Dataset"
3. Select your downloaded CSV file
4. Dashboard will automatically load and visualize the data

## üé§ Viva Questions & Answers

### Basic Level Questions

**Q1: What is Streamlit and why did you choose it?**
**A:** Streamlit is an open-source Python framework for creating web applications. I chose it because:
- Easy to learn and implement
- Requires minimal code for UI components
- Built-in support for data visualization
- Fast prototyping capabilities
- No HTML/CSS/JavaScript knowledge needed

**Q2: How does your dashboard handle missing values?**
**A:** The dashboard uses multiple strategies:
- `dropna()` for critical columns like Age_Group and Screen_Time_Hours
- `fillna()` with default values for non-critical columns
- Data validation to remove invalid entries (e.g., negative hours, >24 hours)

**Q3: Explain the difference between bar chart and histogram in your dashboard.**
**A:** 
- **Bar Chart**: Shows categorical data (e.g., average by age group). Each bar represents a category.
- **Histogram**: Shows frequency distribution of continuous data (screen time hours). Bars represent ranges/bins.

**Q4: What is the purpose of the @st.cache_data decorator?**
**A:** It caches the data loading function to improve performance. Data is loaded once and reused until the file changes, making the dashboard faster.

**Q5: How do filters work in your dashboard?**
**A:** Filters use Streamlit's selectbox widgets. When a user selects a value, the DataFrame is filtered using pandas conditional statements, and all visualizations automatically update with the filtered data.

### Advanced Level Questions

**Q11: Explain the concept of reactive programming in Streamlit.**
**A:** Streamlit uses reactive programming where the entire script reruns when user input changes. This creates a data flow where:
- User interaction triggers rerun
- Cached functions skip recomputation
- UI updates automatically
- State management through session_state

## üìß Troubleshooting

### Common Issues:

**Issue 1: ModuleNotFoundError**
```bash
Solution: pip install [missing_module]
```

**Issue 2: Dashboard not opening**
```bash
Solution: Check if port 8501 is free, or use:
streamlit run screentime_dashboard.py --server.port 8502
```

**Issue 3: Upload file not working**
```bash
Solution: Increase upload limit in .streamlit/config.toml:
[server]
maxUploadSize = 200
```

**Issue 4: Slow performance**
```bash
Solution: 
- Reduce data size
- Use data sampling
- Check @st.cache_data is working
```
